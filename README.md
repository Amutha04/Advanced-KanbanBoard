# Advanced Kanban Board

A modern **Kanban Task Management Web Application** that helps users organize tasks using a visual board with **Todo, In Progress, and Done columns**. Tasks can be created, edited, deleted, dragged between columns, and filtered by priority.

Live Demo: [https://your-kanban-link.vercel.app](https://advanced-kanban-board-tau.vercel.app/)

---

## Features

- Add new tasks with title, description, due date, and priority
- Edit and delete existing tasks
- Drag and drop tasks between columns
- Priority filtering (Low / Medium / High)
- Sort tasks by due date
- Dark mode toggle
- Data persistence using Local Storage
- Clean and responsive UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- LocalStorage (for data persistence)
- Vercel (for deployment)
